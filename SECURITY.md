# Security Policy

## Overview

scroll-zoom-thing is a static documentation site with no server-side code, no authentication, and no user data collection.

## Scope

Because this project is pure CSS and static HTML/Markdown, the attack surface is minimal. Security issues worth reporting include:

- Malicious content injected via docs or templates
- Supply chain issues with MkDocs Material or pip dependencies
- Anything in the Claude agent skills that could cause harm if executed

## Reporting a vulnerability

If you discover a security issue:

1. **Do not** open a public GitHub issue
2. Email the maintainer or open a private security advisory on GitHub
3. Include steps to reproduce
4. Allow reasonable time for a response before public disclosure
