# Upgrading

This guide covers upgrading PAI between versions.

## General Process

1. Read [CHANGELOG.md](CHANGELOG.md) for breaking changes.
2. Back up your user data.
3. Download the new ISO from [releases](https://github.com/nirholas/pai/releases/latest).
4. Flash to a new USB or update in place.

## Version-specific migrations

See [MIGRATION.md](MIGRATION.md) for detailed migration steps between specific versions.

## Rollback

Keep the previous ISO until you've verified the upgrade. To roll back, re-flash the previous version.
