# Citing PAI

If you use PAI in a paper, talk, technical report, or other published
work, please cite it. Accurate citation helps readers find the software
and helps the project justify continued maintenance.

A machine-readable version of this metadata lives alongside this file in
[CITATION.cff](https://github.com/nirholas/pai/blob/main/CITATION.cff). GitHub's "Cite this repository" button,
Zotero, and other citation tools will read that file directly.

## BibTeX

```bibtex
@software{pai,
  title        = {PAI},
  author       = {nirholas},
  year         = {2025},
  month        = {4},
  version      = {0.1.0},
  url          = {https://github.com/nirholas/pai},
  license      = {GPL-3.0-or-later},
  note         = {Released 2025-04-17}
}
```

## APA

> nirholas. (2025). *PAI* (Version 0.1.0) [Computer software].
> https://github.com/nirholas/pai

## IEEE

> nirholas, "PAI," version 0.1.0, Apr. 17, 2025. [Online]. Available:
> https://github.com/nirholas/pai
